웹프로그래밍 수업 후기

1. 수업을 듣게 된 동기

  • 웹사이트 제작에 대한 관심
  • 웹의 동작 원리 이해
  • HTML · CSS · JavaScript 기초 학습
  • 프로젝트 및 교육 콘텐츠 제작에 활용

웹사이트를 통해 정보를 공유하고 다양한 서비스를 제공할 수 있다는 점에 관심이 많았습니다. 직접 웹페이지를 제작하며 웹의 동작 원리를 배우고 싶었고, HTML, CSS, JavaScript를 활용한 웹 개발 기초를 익혀 교육 콘텐츠와 다양한 프로젝트에 활용하고자 웹프로그래밍 수업을 수강하게 되었습니다.

2. 학습 과정

HTML을 이용하여 웹페이지의 구조를 만들고, CSS를 이용하여 디자인을 적용하는 방법을 배웠습니다. 또한 JavaScript를 활용하여 간단한 동적 기능을 구현하면서 웹페이지가 전반적으로 동작하는 원리를 이해할 수 있었습니다.

3. 학습 결과

여러 개의 페이지를 연결한 웹사이트를 직접 제작할 수 있게 되었으며, HTML, CSS, JavaScript의 역할과 활용 방법을 이해하게 되었습니다. 또한 내비게이션 바를 만들고 다양한 레이아웃을 구성하면서 웹페이지 제작에 대한 자신감을 얻었습니다.

4. 느낀점

처음에는 웹페이지를 만드는 과정이 어렵게 느껴졌지만, 하나씩 구현하면서 결과가 화면에 나타나는 것이 매우 흥미로웠습니다. 앞으로는 더욱 다양한 기능과 디자인을 적용하여 완성도 높은 웹사이트를 제작해 보고 싶습니다. 이번 수업을 통해 꾸준한 실습과 반복 학습의 중요성을 느낄 수 있었습니다.
